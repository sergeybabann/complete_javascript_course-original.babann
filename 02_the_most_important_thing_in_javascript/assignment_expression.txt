// Выражение с побочными действиями не только возвращает значение, но и выполняет другие действия


a = 5 
// Кроме возвращения значения (значение 5), приведет к выполнению действия. 
// Действие - значение 5 будет присвоено перменной a  
// Позже в коде можно будет обратиться к этой переменой и прочесть это значение
// То есть были выполнены какие-то действия, кроме возврата значения этого выражения


b++ 
// В этом выражении используется оператор ++ 
// Этот оператор увеличивает значение переменной на единицу. 
// Например, если переменная b содержала цифру 6, то после применения к ней оператора ++ 
// значение этой перемнной будет увеличино на единицу и будет равняться цифре 7
// Побочное действие - увеличение переменной b на единицу


myFunction(c,d) 
// Здесь мы видим вызов функции. Внутри функции мы можем выполнять различные действия.
// Внутри функции выполняются другие побочные действия, кроме возврата значения.
// И так как это выражение, то функция возвращает значение.
